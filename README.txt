Mapeamento:

- Inicialmente o programa começa rasterizando uma linha a cada dois pontos clicados;

l - Muda a forma a ser rasterizada para uma linha a cada dois pontos clicados;
r - Muda a forma a ser rasterizada para um quadrilátero a cada dois pontos clicados;
t - Muda a forma a ser rasterizada para um triângulo a cada três pontos clicados;
p - Muda a forma a ser renderizada para um poligono a cada quatro pontos clicados;
c - Muda a forma a ser renderizada para uma circunferência a cada dois pontos clicados;
f - Muda para o modo preenchimento flood fill, em que, ao clicar em um ponto qualquer, toda a área não delimitada por algum ponto será preenchida. OBS:. Dependendo do tamanho da janela, ao se fora de um poligono pode segmentation fault.
shift + s - Muda a escala da última com os valores informados no terminal;
shift + t - Translada a última figura rasterizada para os valores informados no terminal;
shift + r - Reflete a última figura rasterizada para os eixos informados no terminal;
shift + g - Rotaciona a última figura rasterizada para os valores informados no terminal;
shift + c - Aplica o cisalhamento a última figura utilizando os valores informados no terminal;
